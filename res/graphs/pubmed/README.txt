A citation network whose vertices represent papers, and edges represent
citations between the papers. Each vertex (i.e. publication) is labeled
according to the research area in which it falls.

Information
- fully-connected and undirected
- 19,717 vertices
- 44,324 undirected edges
- 3 labels
- 4,103 vertices with label 0
- 7,875 vertices with label 1
- 7,739 vertices with label 2
- no unlabeled vertices
